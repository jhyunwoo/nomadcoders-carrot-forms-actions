# Next.js Boilerplate

## Features
- Next.js 14
- TypeScript
- Drizzle ORM
- Auth.js (Beta)
- Prettier
- PostgreSQL
- TailwindCSS
- Yarn PnP

## Command Line Interface
### Development
```bash
yarn dev
```

### Build
```bash
yarn build
```

### Start
```bash
yarn start
```

### Lint
```bash
yarn lint
```

### DB Push
```bash
yarn db:push
```

### DB Studio
```bash
yarn db:studio
```
